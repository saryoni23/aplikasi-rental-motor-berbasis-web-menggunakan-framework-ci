Aplikasi Rental Motor 
Aplikasi ini di buat untuk tugas besar matakuliah proyek II.
Matakuliah Proyek II ini dilaksanakan di kampus Politekni Pos Indonesia dengan jurusan DIV Teknik Informatika, 
Matakuliah ini dilakukan dengan cara mengerjakan sebuah aplikasi dan sebuah laporan tentang aplikasi ini.
Aplikasi ini di buat menggunakan bahasa pemograman PHP7 dan menggunakan framework Codeigniter, unutk penyimpanan data menggunakan database yang databasenya menggunakan MYSQL.
